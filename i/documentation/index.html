Setup the app

Install at Local Server

  • Install XAMPP or WAMPP at your localserver
  • Open a directory under htdocs/www folder
  • Unzip downloaded file and upload to that directory
  • Run You XAMPP or WAMPP control Panel
  • Run Apache and MySQL
  • Go to app url: localhost/[your app folder nam]/installs
  • Installation prompt appeared and fill up the details of database
  • App will install automatically
  • After install delete install folder
  • And finally configure your HTACCESS FILE if you installl it in a subfolder
    •             
              RewriteEngine on
              RewriteBase /delivery/   //here delivery is the subfolder name you should change yours one here
              RewriteCond %{REQUEST_FILENAME} !-f
              RewriteCond %{REQUEST_FILENAME} !-d
              RewriteRule ^(.*)$ index.php/$1 [L]
  • Default Login User: admin password: admins

Default User

A default user will be created when the database migrations run for the first time.
This User will have the below login Credentials:
Username: admin
Password: admin
You can edit the default user by through the Admin Panels Users section after logging in.
